# Componentes Comuns

Este diretório contém componentes reutilizáveis que podem ser usados em todo o sistema.

## Exemplos de componentes comuns

- Botões (Button.vue)
- Modais (Modal.vue)
- Inputs (Input.vue, Select.vue, etc)
- Cards (Card.vue)
- Tabelas (Table.vue)
- Loading (Spinner.vue, Skeleton.vue)
- Notificações (Toast.vue, Alert.vue)
- etc.

## Convenções

- Componentes devem ser nomeados em PascalCase
- Devem ser genéricos e não conter lógica de negócio específica
- Devem aceitar props para customização
- Devem emitir eventos quando necessário
